# Memory Evals

Tests whether memory is created, consolidated, and retrieved well.

Scope:
- Observation creation quality
- Belief formation quality
- Retrieval ranking and provenance
- Recall from observations vs beliefs
- Conditional / procedural memory once added

Out of scope:
- Whether the agent prompt uses already-retrieved context correctly
- Transport burst handling
- PMS adapter side effects

Existing `evals/memory-graph/` gold files belong under this system.
