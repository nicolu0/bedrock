# Burst / Session Evals

Tests how inbound messages become agent turns.

Scope:
- Message burst coalescing
- Typo-then-correction handling
- Queued turns while a prior turn is running
- Revalidation before action
- Session boundaries and open-work-order candidate selection over time

Out of scope:
- Tool body voice quality
- Memory retrieval quality
- AppFolio side effects
